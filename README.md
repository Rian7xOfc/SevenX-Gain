# 🚀 SevenX Gain - V1.0

Meu primeiro plugin VST3 desenvolvido com C++ e a framework JUCE.

## ✨ Características
- Controle de ganho preciso.
- Interface Dark Mode personalizada.
- Knob rotativo estilo moderno.

## 🛠️ Como compilar
1. Abra o arquivo `SevenX Gain.jucer` no Projucer.
2. Exporte para o Visual Studio 2022/2026.
3. Compile em modo Release.
