# **🚀 SevenX Gain \- V1.1 (Lançamento Oficial)**

O **SevenX Gain** é um plugin VST3 de alto desempenho desenvolvido em C++ utilizando a framework JUCE. Esta versão 1.1 traz melhorias significativas tanto em funcionalidade quanto em design visual.

## **✨ Novas Características (v1.1)**

* **Controle de Ganho Preciso**: Ajuste de volume linear suave com exibição numérica arredondada.  
* **Saturação Vocal Premium**: Algoritmo de saturação assimétrica otimizado especificamente para dar brilho e corpo a vozes.  
* **VU Meter em Tempo Real**: Medidores de nível RMS (Esquerdo/Direito) para monitoramento visual do sinal de áudio.  
* **Botão de Bypass**: Função de desvio instantâneo para comparação rápida do sinal processado vs. original.  
* **Interface Dark Mode**: Design moderno com elementos em Neon Ciano e feedback visual colorido para os botões.

## **🛠️ Tecnologias Utilizadas**

* **C++**: Linguagem principal para processamento de baixo nível.  
* **JUCE Framework**: Utilizada para gerenciamento de áudio, interface gráfica e compatibilidade VST3.  
* **DSP (Digital Signal Processing)**: Algoritmos customizados de saturação não-linear.

## **🚀 Como Compilar e Usar**

1. Clone este repositório em sua máquina.  
2. Abra o arquivo SevenX Gain.jucer no **Projucer**.  
3. Certifique-se de que os módulos necessários do JUCE estejam configurados.  
4. Exporte o projeto para o **Visual Studio 2022 ou 2026**.  
5. No Visual Studio, altere a configuração para **Release** e compile (**Build Solution**).  
6. O arquivo .vst3 será gerado na pasta de saída (geralmente em C:\\Program Files\\Common Files\\VST3).
